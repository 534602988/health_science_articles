                                    肿瘤化疗药物对心血管系统的影响

                            <p style="text-align:center"><strong><span style="color: rgb(0, 0, 0);"><img src="http://n2.hdfimg.com/g10/M03/7F/12/vYYBAFnpfAmAZasqAABjQolAcmM557.jpg?_ms_=68a5&amp;_flag_=2" alt="<a href=" https:="" m.haodf.com="" citiao="" zhiliao-hualiao.html"="" style="color:#4ba2ed;text-decoration:underline;">化疗.jpeg" style="max-width: 814px !important;" title="1508473865151739.jpeg"/&gt;</span></strong></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">日前，苏沪皖MDT（multi disciplinary team）论坛特别邀请了<strong>李殿富心血管团队</strong>参加，由我代表李殿富教授做了关于《肿瘤化疗药物对心血管的影响》的专题报告。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">随着治疗手段的日新月异，化疗、放疗和靶向药物治疗，使恶性肿瘤的缓解率和治愈率明显提高，新的概念认为恶性肿瘤如同<!--HAODF:8:gaoxueya-->高血压<!--HAODF:/8:gaoxueya-->、<!--HAODF:8:tangniaobing-->糖尿病<!--HAODF:/8:tangniaobing-->一样，是一种慢性病，人们将不再谈癌色变了。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">但是新的矛盾也由此产生了：肿瘤的化疗放疗和靶向药物对心血管的影响，已不容小觑，尤其是<strong>化疗和靶向药物有可能导致<!--HAODF:8:gaoxueya-->高血压<!--HAODF:/8:gaoxueya-->、心肌缺血、心力衰竭</strong>。其中心力衰竭的5年生存率只有20%，比很多恶性肿瘤如前列腺癌、乳腺癌和结肠癌的死亡率还高。这种现象已经引起了肿瘤学界高层专家的高度重视，如何在治疗肿瘤的同时，尽量减少对心血管系统的影响，已经成为了医学界的重要课题。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><strong><span style="color: rgb(0, 0, 0);">对于肿瘤患者来说，既不能因噎废食，放弃肿瘤的治疗；同时也要加强防范，尽可能减小抗肿瘤药物对心血管系统的伤害。</span></strong></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">化疗药物的心脏毒性与多种因素有关。比如与化疗药物本身、每次给药的剂量及累积剂量，给药的时间、途径、联合用药的情况及给药的顺序均有关系；而且与患者的年龄、合并的基础心血管疾病以及药物治疗有关。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><strong style="text-indent: 2em;"><em>常见的化疗药物对心血管的损伤：</em></strong></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">1、蒽环类：用于治疗多种血液系统肿瘤和实体瘤，最常用于乳腺癌，如阿霉素、表柔比星、柔红霉素，其心脏急性毒性为非特异性ST段及T波异常，迟发的心脏毒性主要是心力衰竭，发生率2%/年，与剂量相关，剂量越大，心衰发生率越高。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">2、曲妥单抗：用于Her阳性的乳腺癌的靶向治疗，单用心力衰竭发生率为3-5%，合用蒽环类药物心力衰竭的发生率在16%左右。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">3、酪氨酸激酶抑制剂：伊马替尼、达沙替尼、尼罗替尼多用于白血病的治疗，心脏损伤的发生率为1%，舒尼替尼和索拉非尼心脏毒性的发生率为3%。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">4、抗体治疗：主要是发生<!--HAODF:8:gaoxueya-->高血压<!--HAODF:/8:gaoxueya-->，比如贝伐单抗<!--HAODF:8:gaoxueya-->高血压<!--HAODF:/8:gaoxueya-->的发生率是50%，而西妥昔单抗和利妥昔单抗容易出现体位性低血压。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">5、抗代谢治疗：主要药物为卡洛他宾和5-氟尿嘧啶，可导致1.2%-1.8%的患者出现心肌缺血。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">6、烷化剂：顺铂可导致<!--HAODF:8:gaoxueya-->高血压<!--HAODF:/8:gaoxueya-->，一些老年患者可能出现心肌缺血、血栓形成、急性冠脉综合征和症状性左心室功能不全；环磷酰胺多用于治疗血液系统的恶性肿瘤，常产生心肌炎，尤其是剂量大于150mg/kg，发生率在7-25%。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">7、紫杉烷类：多西他赛和紫杉醇多用于治疗乳腺癌、<!--HAODF:8:feiai-->肺癌<!--HAODF:/8:feiai-->和卵巢癌，导致心脏毒性的比例为2.3-8.0%。</span></p><p style="text-align:center"><span style="color: rgb(0, 0, 0);"><img src="http://n2.hdfimg.com/g10/M03/7E/F9/voYBAFnpe_aAeXPIAAE11iLn8UE944.jpg?_ms_=1381&amp;_flag_=2" alt="防心脏病.jpeg" style="max-width: 814px !important;" title="1508473846172056.jpeg"></span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><strong><em><span style="color: rgb(0, 0, 0);">加强监测，早期发现化疗导致的心血管损伤，及早治疗：</span></em></strong></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">1. 化疗期间应加强症状、体征、心电图和心脏超声的监测，但是一旦出现心电图和心脏超声的明显异常，患者病情一般已比较严重，所以探索早期的诊断方法非常重要，比如心脏超声对心肌变应力的检测，结合血清肌钙蛋白T和脑利钠肽的检查，做到早期及时的诊断是提高预后的最佳方法。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">2. 暂停化疗和靶向药物治疗。</span></p><p style="margin-bottom: 20px; line-height: 1.65em; font-family: 微软雅黑, Helvetica; color: rgb(51, 51, 51); white-space: normal; text-indent: 2em; text-align: left;"><span style="color: rgb(0, 0, 0);">3.
