                </div><p>先天性心脏病，是胚胎发育早期（怀孕2-8周），由于某种原因造成的心血管结构或功能的异常，其发病率约占0.6%-1%，也就是说每100名新生儿中就有一名患有先天性心脏病，是儿童最常见的出生缺陷疾病，我国每年约有15万-20万的先心宝宝出生。</p><p>先天性心脏病谱系特别广，包括上百种具体分型，有些患者可以同时合并多种畸形，症状千差万别，最轻者可以终身无症状，重者出生即出现严重症状如缺氧、<span style="color:rgba(0, 0, 0, 0.847)">休克</span>甚至夭折。根据血液动力学结合病理生理变化，先天性心脏病可分为发绀型或者非发绀型，也可根据有无分流分为三类：无分流类（如<span style="color:rgba(0, 0, 0, 0.847)">肺动脉狭窄</span>、<span style="color:rgba(0, 0, 0, 0.847)">主动脉缩窄</span>）、左至右分流类（如<span style="color:rgba(0, 0, 0, 0.847)">房间隔缺损</span>、<span style="color:rgba(0, 0, 0, 0.847)">室间隔缺损</span>、<span style="color:rgba(0, 0, 0, 0.847)">动脉导管未闭</span>）和右至左分流（如法洛氏四联症、大血管错位）类</p>
